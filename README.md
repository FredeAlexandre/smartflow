# Smartflow

Smartflow is a no-code plateforme to create, build & deploy smart contracts.

## Getting Started

### Requirements

- [Bun](https://bun.sh/)

### Instalation

1. Clone the repository

```bash
git clone git@github.com:8pes/smartflow.git
```

2. Install javascript dependencies

```bash
cd smartflow
bun install
```

3. Start developement server

```bash
bun dev
```
